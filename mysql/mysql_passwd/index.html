mysql/mysql_passwd

Posted on 2021-06-11

mysql root 密码修改

服务器配置 ：centos 8.0

安装 ：mysql完整版

需要的可以到mysql官网下载 ： 地址

主要记录mysqlroot 密码修改 因为本人第一次登陆的时候并没有填写密码，所以在/var/log/mysql/mysqld.log 里面没有记录默认密码

mysqld.log 的位置不一定相同 具体请使用whereis mysqld.log 查看

修改过程

首先查看 /etc/my,cnf 配置文件 跟别人并不相同

没有更多限制

直接使用 mysql可以进行登陆

推荐使用mysql -uroot -p

在你的mysql.log 没有密码时 直接回车键登陆

登陆后执行

alter user ’root’@’lcalhost’ identified by ‘your password’ ;

执行成功

下面再使用mysql 会出现报错

因为这时mysql root已经设置密码

所以需要使用 mysql -uroot -p 登陆，并输入密码

登陆成功
